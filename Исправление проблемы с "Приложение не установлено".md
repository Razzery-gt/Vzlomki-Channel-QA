## Введение
Каждый, кто скачивал приложения через APK сталкивался с очень обидной прлблемой, что приложение может не устанавливатся. Это может происходить из за множества различных причин.
### Причины этого
1. Конфликт версий
2. Конфликт подписей
3. Архитектура процессора (пример: вы пытаетесь скачать 64 битное приложение на 32 битное устройство) 
4. Версия Андроид не подходит для этого приложения (пример: вы пытаетесь скачать приложение для андроид 5 на андроид 4) 

## Как решить эту проблему (или хотя-бы разобраться почему она возникает) 

1. Скачивайте Shizuku и Install With Options с Телеграм канала @AndroidUtils 
2. Запускайте службу Shizuku (инструкция есть в приложении) 
3. После запуска службы Shizuku заходите в приложение Install With Options 
4. В Install With Options выбирайте ваш файл и начните установку. 
Наверняка после этого шага вы столкнетесь с ошибкой. Это нормально, она вам и нужна, так как там есть специальная отладочная информация которая поможет вам разобраться с проблемой.
